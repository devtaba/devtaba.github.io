Port of Seattle

I interned at the Port of Seattle the summers of 2018 and 2019. There, I worked on solving problems at the SeaTac International Airport with machine learning. Most notably, I helped build a solution for detection of large luggage on airport escalators to aid in mitigating the rise in airport escalator accidents. Using this detection model, I was also able to provide an analysis of indoor traffic patterns at the airport. This solution was also featured in the International Airport Review in 2020. You can check out the paper documenting my work here. My work utilized Tensorflow, Numpy, Pandas, Jupyter Notebook, and annotations were done using labelImg

Librocast

Librocast is a full-stack web application that allows users to create a profile where they can keep track of reading goals, books they've read, want to read, and are currently reading, as well as keep up with friends. Overall, the goal of this app is to create a modern social media for readers and incentivize reading in today's era. You can check it out on GitHub here. The project uses React.js on the frontend and Node.js, Express, and, Firebase on the frontend. I also prototyped the design on Figma. Testing was done using Jest and React Testing Library.

You can read my research report here.

Campus Maps

Campus Maps is a full-stack web application that allows users to find the shortest path between two buildings at the UW campus using Dijkstra's pathfinding algorithm. The frontend is built in Typescript, React.js and the backend uses a custom Graph data structure built in Java as well as the Spark Java framework. Check out the code on GitHub here.

Barber shop Booking App

Inspired by my younger brother's passion for cutting hair, I built a web app for booking barber services. The app provides authentication, booking availability and appointment setting, and payments for services and security deposits. The frontend uses React.js and MaterialUI. The backend uses Node.js, Express and Firebase. You can check the code out on GitHub here.
